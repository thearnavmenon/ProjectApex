PROGRAM STRUCTURE
═══════════════════════════════════════════════════════════════
• Number of training days per week: EXACTLY programming_constraints.training_days_per_week.
• Each session: 4–6 exercises. Compounds first, accessories after.
• Use consistent day_labels across all phases (e.g. Push_A must appear in all four templates on the same day_of_week).
• Balance push/pull/legs volume across the week.
• Deload uses the same day_labels but reduces sets by ~50% and raises RIR to 4–5.

═══════════════════════════════════════════════════════════════
EXERCISE VARIATION ACROSS PHASES
═══════════════════════════════════════════════════════════════
Exercises MUST rotate across the four phases — do NOT repeat the same exercise selection
in every phase. Use the phase goals below to drive variation:

ACCUMULATION (weeks 1–4):
  • Higher rep ranges (8–15), moderate load. Prioritise exercise variety.
  • Use a wider range of movements — e.g. DB Incline Press, Cable Flyes, Hack Squat,
    Romanian Deadlift. Build movement patterns and muscle familiarity.
  • Include one higher-rep accessory per session (12–15 reps).

INTENSIFICATION (weeks 5–8):
  • Heavier compounds, reduced variety. Shift toward barbell-dominant movements
    where available — e.g. Barbell Bench, Barbell Row, Back Squat, Overhead Press.
  • Drop or replace the higher-rep accessories with a second compound variation.
  • Rep ranges narrow (5–10). Progressive overload is the priority.

PEAKING (weeks 9–11):
  • Minimum exercise selection — 4 per session maximum. Primary compound movements only.
  • e.g. Bench Press, Squat, Deadlift, Overhead Press, Pull-Up. No accessories.
  • Volume drops sharply; intensity peaks. RIR 1–2.

DELOAD (week 12):
  • Full exercise rotation reset — return to accumulation-style variety, lighter loads.
  • Technique focus. Use exercises that differ from both the peaking phase AND the
    original accumulation phase where possible (novel movements are fine here).
  • RIR 4–5, sets ~50% of peak week.
